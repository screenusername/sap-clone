# SAP Clone

A Super Auto Pets–style auto battler built with Phaser 3.

## Version 0.1 — Battle mock-up

Static battle view with 3 pet slots per side, placeholder pets (Ant, Baboon, Cat vs Dog, Elephant, Fox), and attack/defense stats with sword/shield icons.

## Battler sprites

Add transparent PNGs to `public/assets/battlers/` — see that folder's README for naming and sizing.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
```
