# Battler sprites

Drop **transparent PNG** files in the project `assets/battlers/` folder (at the repo root). Vite copies them into the build automatically.

## File naming

Use the **lowercase pet name**:

| Pet      | Filename        |
|----------|-----------------|
| Baboon   | `baboon.png`    |
| Cat      | `cat.png`       |
| Dog      | `dog.png`       |
| Elephant | `elephant.png`  |
| Fox      | `fox.png`       |
| Goth     | `goth.png`      |
| Nun      | `nun.png`       |
| Punk     | `punk.png`      |
| Slime    | `slime.png`     |
| Wolf     | `wolf.png`      |

## Recommended source size

- **Format:** PNG with transparency
- **Suggested canvas:** about **500×760 px** (or any size matching a **5:7.6** width-to-height ratio)

## Notes

- You can add sprites one at a time; pets without a matching file still show the card (name + stats only).
- After adding or replacing files, refresh the browser to reload.
